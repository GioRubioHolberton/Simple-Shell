# SIMPLE SHELL

Simple shell that prints some linux commands.

## How to star

1. Clone the repository
git clone https://github.com/GioRubioHolberton/simple_shell.git

2. Compilation process
gcc -Wall -Werror -Wextra -pedantic * .c -o hsh

3. Execute
./hsh
